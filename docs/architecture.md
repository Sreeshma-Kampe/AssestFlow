# AssetFlow Architecture

## Overview
AssetFlow is an enterprise asset and resource management platform designed for internal operations, maintenance planning, and lifecycle tracking.

## Stack
- Next.js 15 App Router
- TypeScript
- Tailwind CSS
- Prisma ORM
- PostgreSQL
- JWT Authentication
- Local file storage

## Core Areas
- App shell and dashboard layout
- Auth and user management
- Asset and resource modules
- File and document storage
- Reporting and audit trails

## Directory Shape
- app/: route groups and pages
- components/: reusable UI and layout elements
- lib/: shared utilities and helper modules
- prisma/: data model and database configuration
- storage/: local file handling
